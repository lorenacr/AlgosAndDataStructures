# AlgosAndDataStructures

This project will be used to exercise concepts about algorithms and data structure such as:
- Stacks
- Queues
- Linked lists
- Binary trees
- Hash tables


___
#### Disclaimer
The goal of this project is not performance, it is about practicing and fixing concepts.
